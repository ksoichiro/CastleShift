All notable changes to this project will be documented in this file.

## [Unreleased]

## [0.2.0] - 2026-03-13

### Added
- Forge support
- Minecraft 1.21.2, 1.21.3, 1.21.4, 1.21.5, 1.21.6, 1.21.7, 1.21.8, 1.21.9, 1.21.10, 1.21.11 support

### Fixed
- Castles no longer generate too close to villages

## [0.1.0] - 2026-02-22

### Added

#### Structures
- Castle structure with multi-floor layout (floor1-floor4)
- Corner, wall, and center room structures with multiple variants
- North wall tower structures
- Barrel loot table for castle structures

#### Processors
- Roof material random replacement with dark prismarine, red nether brick, and crimson variants
- Wall material random replacement
- Stair material random replacement
- Wall weathering for stone brick and polished blackstone brick walls
- Material combination system to coordinate roof and wall material selection

#### Minecraft Version Support
- Minecraft 1.21.1
- Minecraft 1.20.1

#### Multi-Loader Support
- Fabric support for Minecraft 1.20.1, 1.21.1
- NeoForge support for Minecraft 1.21.1
